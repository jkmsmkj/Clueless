# Clueless Game
A MATLAB app for [Problem 61069 of Cody Contest 2025: Clueless - Lord Ned in the Game Room](https://www.mathworks.com/matlabcentral/cody/groups/96682/problems/61069).

## Features
1. Solve the 1+20 games (test cases) given along with the problem step-by-step.
2. Play an unscripted game and choose between three levels of difficulty.
3. Contribute to the source code to make the app better!
